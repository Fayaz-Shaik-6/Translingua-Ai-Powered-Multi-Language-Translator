# 🌐 **Translingua – AI Powered Multi-Language Translator**

## 📘 Project Overview
Translingua is an AI-powered multi-language translator that allows users to input text and get translations in multiple languages. [scribd](https://www.scribd.com/document/960432634/Source-Code)
The project uses **Streamlit** for an interactive web interface and is structured to separate code, documentation, and demo files for clarity. [scribd](https://www.scribd.com/document/960432634/Source-Code)

## 🧱 Project Structure

```bash
Project_Files/
    app.py           # Main application script
    translang.py     # Translation logic module (Streamlit app)
    requirements.txt # Required Python packages

Documentation/       # Contains project documentation

Video Demo/          # Contains demonstration videos showcasing the project in action

.gitignore           # Ensures unnecessary files are not uploaded
```

## 🚀 How to Run

Set up and run the project step by step:

```bash
# Create a virtual environment
python -m venv venv

# Activate the environment
source venv/bin/activate      # Mac/Linux
venv\Scripts\activate         # Windows
```


```bash
# Install dependencies
pip install -r Project_Files/requirements.txt
```


```bash
# Run the main app script
python Project_Files/app.py
```


```bash
# Or launch the interactive Streamlit app
streamlit run Project_Files/translang.py
```
